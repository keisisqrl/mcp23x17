    init_driver([term()]) :: Supervisor.on_start_child()

    Create a Mcp23x17.Driver in the supervision tree and return it.

    Always call this with the base (7-bit) address of the chip. This can be calculated from the A pins using Mcp23x17.Utils.base_addr/1.

    Examples 

    iex> Mcp23x17.init_driver([33, nil, nil, Mcp23x17.Adapters.Mock])
